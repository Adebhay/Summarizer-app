# 🧠 Mentiis.co - AI Productivity Coach

**Mentiis.co** is a cross-browser extension that tracks your daily browsing activity and provides AI-powered productivity insights and coaching.

## ✨ Features

- 📊 **Track Activity** - See what websites you spend time on
- 🤖 **AI Insights** - Get personalized daily coaching summaries
- 🎯 **Actionable Tips** - Improve your daily focus
- 🔒 **Privacy First** - Your data stays with you
- 🌐 **Cross-Browser** - Works on Chrome, Firefox, and Edge

## 📥 Download

Download the extension from the [Releases](https://github.com/Adebhay/Summarizer-app/releases/tag/v1.0.0) page.

## 📋 Installation

### Chrome / Edge
1. Download the `.crx` file
2. Go to `chrome://extensions/` (Chrome) or `edge://extensions/` (Edge)
3. Turn ON "Developer mode"
4. Drag the `.crx` file into the browser
5. Click "Add extension"

### Firefox
1. Download the `.zip` file
2. Extract the zip
3. Go to `about:debugging`
4. Click "This Firefox"
5. Click "Load Temporary Add-on"
6. Select `manifest.json`

## 🧪 How to Test

1. Browse websites for 5-10 minutes
2. Click the Mentiis.co icon in your toolbar
3. Click "Get AI Summary"
4. Get your personalized coaching summary!
